Gamania
Welcome to Gamania, a comprehensive, Wikipedia-style encyclopedia dedicated to all things gaming.

About
Gamania is a reference project designed to serve as a hub for game genres. Users can explore various genres, learn about their unique histories, and discover details about specific titles, including:
- Pricing
- Player count
- Release dates

How to Use
- Explore: Browse through the list of game genres to learn about their evolution and key characteristics.
- Discover Games: View lists of games within each genre. Clicking on any individual game title will redirect you to its official Fandom page for more information.
- Access the Website: You can visit the live site here: https://dodorex23.github.io/Gamania/
- Download Files: If you prefer to view the source code locally, click the green Code button at the top of this repository and select Download ZIP.

Technologies Used
- HTML5: Structural markup for all pages.
- CSS3: Styling and layout design.
- Assets: Custom image support including .png and .jpg formats.
- Zero JavaScript: This project is built entirely using static web technologies.
